# Introduction

Welcome to the sheweny guide, this part of the site gives explanations and examples of uses of certain points of the framework. If you wish to consult the documentation click here

In this guide you will find :

  • How to install the framework and its dependencies
  • What project structure creates
  • How to instantiate and configure the client
  • How to set up the managers
  • How to create commands, context-menus, buttons, select-menus, events...
  • How to deal with errors

This guide tries to be as clear and fair as possible, but it still needs to be improved. If you want to contribute you can open an issueopen in new window or create a pull requestopen in new window

If you want help with a problem, access the latest announcements or even join the community you can join the discord serveropen in new window.
