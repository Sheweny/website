# As easy as that

```shell-session

# Install cli

npm install -g @sheweny/cli

# Create the bot

sheweny create

# Start the bot

npm start
```

_Note : Your are on the V3.0.0 documentation, for the V2.0.0 documentation please go to [this website](https://sheweny.github.io/V2-website/)_
